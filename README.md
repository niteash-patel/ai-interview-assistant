# AI Interview Assistant

An AI-powered mock interview platform with voice interaction.

## Features
- Voice-based interview (speak your answers)
- AI-generated questions based on job role
- Input validation
- End-of-interview evaluation report
- Professional UI

## Tech Stack
- Python, Streamlit
- OpenAI GPT + Whisper
- gTTS (Text to Speech)

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add `.env` file with `OPENAI_API_KEY=your_key`
4. Run: `streamlit run app.py`
